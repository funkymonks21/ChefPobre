# ChefPobre

O Trabalho final da disciplina de Desenvolvimento Web I, no curso de Análise e Desenvolvimento de Sistemas. Website feito com HTML e CSS puro.

## Ideia do site

A ideia principal do site é ser um repositório de receitas de baixo custo, onde os próprios usuários possam mandar receitas e avaliar receitas de outros usuários.
As receitas são dividas por tipo de refeição (almoço, lanche, café da manhã) ou por ingredientes principais (frango, carne bovina, vegetariano) e também podem ser incluídos em categorias de restrição (sem glúten, sem lactose).

## Processos antes do código

Antes de começar a escrever o código do site de fato, eu fiz pesquisas e esboços de como seria o layout principal do site, quais seriam as páginas criadas, quais seriam os formulários e desenhei um fluxo.
